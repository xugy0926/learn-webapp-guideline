# 介绍

该开发教程是基于 Node/Express 的 WebApp 教程，旨在让初学者快速学习开发一个前端和后端一体的 WebApp。

本教程希望学习者有一些 JavaScript 的基础。教程中会涉及到很多新的概念和开发模块，对于初学者来说，理解上会很难。但是这都没有关系，初学者只需要按照教程一步一步的去做，就一定会做到最后的结果。达到结果后，再回过头去摸索教程中提到的所有概念。

祝学习愉快。
